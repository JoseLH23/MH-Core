# MH Core - Blueprint General

## Visión

Construir un ecosistema de software modular llamado MH Core, capaz de soportar múltiples aplicaciones y servicios.

Las primeras aplicaciones serán:

- MindHigh
- EjiXhole

En el futuro podrán añadirse nuevos proyectos sin modificar el núcleo.

---

# Objetivo

Crear una plataforma sólida, escalable y profesional sobre la cual desarrollar aplicaciones móviles, web y de escritorio.

# Filosofía

No construir aplicaciones aisladas.

Construir un núcleo reutilizable que permita crear cualquier proyecto futuro.

Cada nueva aplicación deberá conectarse al MH Core.
